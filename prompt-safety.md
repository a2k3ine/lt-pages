# AIにさせないこと（禁止事項）

AIがファイル削除や環境変更をしないようにするための安全ルールです。

CursorはAIがOSコマンドを勝手に実行できない設計のようですが、過去に Claude Code や Antigravity でファイルが消えた事例もあると聞きました。

そのため、プロンプトの中で「削除・上書き・環境変更をしない」ルールを明示的に追加しました。

***

【禁止事項】

AI がファイル保存してはいけない

Windows のパスへ直接書き込み禁止

カレントディレクトリ操作禁止

「実行」「Run」などの命令文をプロンプト内に書かない（このテンプレートには既に書かれているので増やさない）

JSON や中間データをそのまま出力しないこと
（このプロンプトの出力は Markdown ノート + save_suggestion のみ）
